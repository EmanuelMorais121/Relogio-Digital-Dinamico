O projeto do relógio digital é um aplicativo web desenvolvido em HTML, CSS e JavaScript. Fornece uma interface de relógio digital que exibe a hora atual.

Exibe a hora atual com precisão, atualizando em tempo real para nos dar a hora, os minutos e os segundos atuais.
